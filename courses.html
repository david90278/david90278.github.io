Courses

I have been working on brushing up my coding skills since 2017. This section lists some courses I have taken and what I learned from each of the courses. I will also list my own courses here as I publish them online.

JavaScript

I started my JavaScript journey with Wes Bos' JavaScript 30 course. This course covers 30 challenges using only vanilla JavaScript with no frameworks. From there I signed up for his Beginner JavaScript course.

Python

My first Python course was The Complete Python Bootcamp from Udemy. It provided a solid background for understanding Python and using Jupyter for projects.

Ruby

My first Ruby course was Learn to Code with Ruby which I took on Udemy. It was a good introduction to the Ruby Programming language.

Swift UI

I am currently 1/3 of the way through 100 Days of Swift UI. This course has given me a good handle on Views as well as how to use Optionals and Closures.
